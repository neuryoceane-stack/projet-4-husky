# Icônes

Placez ici les icônes si nécessaire.

**Fichiers optionnels :**
- Icônes sociales (Facebook, Instagram, etc.)
- Favicon (`favicon.svg` ou `favicon.ico` à placer à la racine de `/public`)
- Autres pictogrammes personnalisés

**Recommandations :**
- Format : SVG de préférence
- Dimensions : 24x24px à 64x64px selon l'usage
- Style : cohérent avec l'identité visuelle du site
